I desgined this 2024 version based on the RepRap Pro Huxley and auctioned off the first and only build of this machine for the Sanjay Mortimer Foundation. It was particularly special since the RepRap Pro Huxley was the 3D printer that got Sanjay into 3D printing and was one of the catalysts for starting E3D.

The original aim was to release a kit (hence a detailed bom is included) but since I was not able to execute this I have released the files on their own without any futher guides and instructions.

I wont be providing any direct support for builds and kits at this time, but I'm happy to discuss and provide some general help in the Vector 3D discord huxley channel where you can also chat with others looking to do the same: https://discord.gg/9nwC26CGGS
